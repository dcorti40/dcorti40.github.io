About Me

My Projects

EV3 Robot

Designed a Lego EV3 robot using Robot C programming to pick up red cups and bring them outside of the robots ultrasonic sensor range. 3D printed claws pick up the cups with lowering mechanisms and the robot will continue its program until all cups are picked up.

Romanoake

Created a web karoake app that romanzies foreign language lyrics (i.e 你好 -> Nǐ hǎo). App uses spotify api and azure speech services to romanize text and play music through the spotify app. Scoring is based on categorizing the music analysis data provided by spotify's API.

RC Car

Created a prototype self driving rc car that uses machine learning algorithms to determine whether or not to turn or not. Uses a rack and pinion steering system with back wheel drive. Program is run through a raspberry pi and uses a motor controller to adjust steering and a 30 FPS camera to continuously update the software.

QuakeMaster

Created a shake table toy similar to jenga quake. Uses a system of cam mechanisms to lift the various tiles of the shake table and move the blocks. Blocks were designed using interconnecting pins to enable creativity amongst users.
